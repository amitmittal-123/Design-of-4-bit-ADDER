# Design-of-4-bit-ADDER(3-Bit Output)

This is a simple digital circuit that adds four 1-bit binary inputs (`A`, `B`, `C`, `D`) and outputs a 3-bit binary sum (`SUM`), ranging from 0 to 4.

## Inputs
- A (1 bit)
- B (1 bit)
- C (1 bit)
- D (1 bit)

## Output
- SUM (3 bits)

## Example
| A | B | C | D | SUM |
|---|---|---|---|-----|
| 1 | 0 | 1 | 1 | 011 |
| 1 | 1 | 1 | 1 | 100 |
